# Thesmal Thesmalrepo

## How do I install these formulae?

`brew install thesmal/thesmalrepo/<formula>`

Or `brew tap thesmal/thesmalrepo` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "thesmal/thesmalrepo"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
